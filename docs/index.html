# Bienvenido a Cogny

Tu nueva forma de tomar notas.

- **Markdown** nativo
- Temas personalizables
- Rápido y ligero

¿Por qué Cogny?

Ultrarrápido

Diseñado para ser ligero y responder al instante. Olvídate de los tiempos de carga.

Personalizable

Temas Drácula, Dark y Light integrados. Cambia los colores a tu gusto.

Privado

Tus notas son archivos locales. Sin nube obligatoria, sin rastreadores.

Descarga Cogny

Elige tu plataforma y empieza a escribir.

Descargar para Windows (.exe) Descargar para Linux (.deb)

O instálalo vía repositorio (Debian/Ubuntu):

Terminal
curl -fsSL https://Maalfer.github.io/cogny/cogny.gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/cogny-archive-keyring.gpg

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/cogny-archive-keyring.gpg] https://Maalfer.github.io/cogny stable main" | sudo tee /etc/apt/sources.list.d/cogny.list > /dev/null

sudo apt update
sudo apt install cogny
